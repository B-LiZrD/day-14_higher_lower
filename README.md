# day-14_higher_lower
Day 14 of 100.


Another program that should be run in repl.it because of the importation of the replit module. Who has more followers, A or B? If the user guesses correctly, slot B gets moved up to A's position, a point is gained, and the game continues. Once the user gets it wrong, the final score is displayed. Here I gained experience working with a list of dictionaries, and accessing various values by calling the associated keys.
